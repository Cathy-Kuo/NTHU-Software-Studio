| Basic rules                 |  20%  |  Y  |

| Jucify mechanisms           |  15%  |  Y  |

[出現的可以拾取的特殊物品，加快速度物品一直都有、自動瞄準物品和穿透物品在level_2後才會出現、小幫手物品在level_3後才會出現、回血物品在level_5後才會出現]
